libm2k Library Setup in MATLAB 2024b

Objectives:

This document will walk through how to set up the libm2k Library in MATLAB and use the library to interface with the ADALM-2000 (M2K) with Windows.

Preparation:

Before you start, please make sure that:

  1. The MATLAB version is 2024b.
  2. The M2k driver is installed on your computer (If so needed, install the driver for Windows here).

Install libm2k Toolbox:

Step 1: Go to https://drive.google.com/file/d/1VVwdFxmQhbZ2zXMJxsF26NKqgWsEUKyc/view?usp=sharing and download the toolbox.

Figure 1: libm2k Toolbox download button

Step 2: Drag the “AnalogDeviceslibm2kBindings_vxx.x.x.mltbx” file to your MATLAB working directory (Figure 2). Double-click on the .mltbx file in MATLAB to install the toolbox. You will see an “installation complete” message after the installation (Figure 3).

Figure 2: Drag the toolbox file to the MATLAB working directory

Figure 3: “Installation Complete” window displayed after toolbox installation

Test the Toolbox:

Connect ADMAL-2000 to your computer. Run the following command in the MATLAB command window.

m2k = clib.libm2k.libm2k.context.m2kOpen();

If the command executes successfully, as shown in Figure 4, you may proceed to the next section.

If not, check the cable connection.

If you see an error message similar to Figure 5, make sure you have installed the toolbox.

If you see an error message similar to Figure 6, make sure the MATLAB version is 2024b or higher.

Figure 4: Command output showing successful toolbox installation

Figure 5: Error message related to the toolbox

Figure 6: Error message related to MATLAB version

Execute the “ReadPowerSupply.m” File:

This section explains how to use the “ReadPowerSupply.m” file to test whether the libm2k library has been set up correctly in your programming environment. You do not need to understand the code in “ReadPowerSupply.m”, it is simply a program for troubleshooting the programming environment.

You only need to run “ReadPowerSupply.m” once during Lab 6 to verify the setup, after which your computer should be ready for all MATLAB M2K programming assignments in future labs and in the final project.

For your future MATLAB M2K programming assignments, you will use the “M2K.m” class to control the ADCs and DACs on the M2K device.

Step 1: Add the “ReadPowerSupply.m” file to the MATLAB working directory.

Step 2: Double-click “ReadPowerSupply.m” to open the file.

Step 3: Connect the ADALM pins (1+, V+, 1-, GND) as instructed in “ReadPowerSupply.m” comment section. Run the file.

If you get a voltage reading around 1.7V like in Figure 7, you may proceed to the Lab 6 instruction, Section 4.2 Procedure.

If you see an error message similar to Figure 5, proceed to step 4 to set up the MATLAB search path.

Figure 7: “ReadPowerSupply.m” program output

Step 4: The error is probably caused by a missing search path for the libm2k library files, which prevents MATLAB from locating the function definition during code execution. To resolve this issue, restart MATLAB and run the “ReadPowerSupply.m” file again. If it still doesn’t work, add the library path to the MATLAB search path by following step 5.

Step 5: To find the local path of the libm2k library, open the Add-Ons Manager by selecting the options boxed in Figure 8 and Figure 9.

Figure 8: Steps to open the Add-On Manager

Figure 9: Steps to open the libm2k folder

Step 6: Double-click on the libm2k folder in the working directory (Figure 10), then copy the file path (Figure 11).

Figure 10: libm2k folder

Figure 11: Path to the libm2k DLL files

Step 7: Open the Set Path window in the MATLAB Home page. Select “Add with Subfolders”, navigate to the path from Step 6, then click “Select folder”. The libm2k DLL file paths should now be added to the search path list. Click “Save” and then close the Set Path window.

Figure 12: Set Path window

Step 8: Navigate to the folder containing the “ReadPowerSupply.m” file and execute the program again to ensure it can run with no errors.

Figure 13: Set Path window
